# Source Material

This folder documents where raw material belongs.

Raw material is useful because not every important moment is obvious while it is happening.

Raw material is also dangerous because it can be noisy, private, or confusing.

## Private Raw Captures

Use:

```bash
make capture-source
```

Default output:

```text
source-material/private/codex-sessions/
```

That folder is ignored by git.

## Public Promotion

Do not publish raw captures.

Promote selected material into:

- `evidence/`
- `RECEIPTS.md`
- `BUILD_LOG.md`
- `CASE_FILE.md`

The public should see the proof, not the entire workspace exhaust.
