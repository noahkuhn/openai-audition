# Launch

This folder holds the public attention assets.

The goal is not to explain everything.

The goal is to make the right people curious enough to inspect the case file.

Primary assets:

- `x-thread-draft.md`
- `video-outline.md`
- `openai-outreach.md`

Conversion rule:

Public attention only matters if it routes to an OpenAI conversation. Every launch asset should point serious readers to `CASE_FILE.md`, `OPENAI_FIT.md`, and `APPLICATION.md`.
