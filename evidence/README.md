# Evidence

This folder is for curated public proof.

It is not for raw session dumps.

Good evidence is:

- specific,
- short enough to inspect,
- linked from a claim,
- reviewed for sensitive information,
- useful to the public story.

Examples:

- selected transcript excerpts,
- screenshots,
- commit references,
- before/after artifact diffs,
- prompt excerpts that changed direction,
- proof that Codex generated or modified a concrete artifact.
