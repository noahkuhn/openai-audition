# Launch Thread Draft

## Post 1

I asked Codex to get me hired at OpenAI.

It told me not to start with a cover letter.

It helped me build a public audition instead: strategy, scorecards, rejected ideas, prompts, build logs, receipts, and the final application.

This repo is the application.

## Post 2

The premise:

Codex should not just help people code faster.

It should help people convert intent into durable public work.

So I gave it a human objective, not a ticket:

Get me hired by OpenAI.

## Post 3

The first thing Codex did was not build.

It forced project selection:

- What would OpenAI care about?
- What would builders share?
- What would prove taste and execution?
- What would be honest enough to inspect?

## Post 4
